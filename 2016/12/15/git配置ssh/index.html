Github配置sshkey

发表于 2016-12-15 | |

github 配置sshkey

使用ssh-keygen命令生成key,提示需要输入密码是可以按回车跳过，这里输入密码表示每次push时都需要输入密码，按下回车后以后每次git push时就不用输入密码了。

s.g.

测试ssh keys是否设置成功

使用ssh -T 命令测试设置是否成功,如果出现

You’ve successfully authenticated, but GitHub does not provide shell access.

表示设置成功
