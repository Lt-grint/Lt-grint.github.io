1.Javascript

下列的值在javascript中表示false

  • false
  • null
  • undefined
  • 空字符串’ ‘
  • 数字0
  • 数字NaN

    逻辑与：&&

    当&&的第一个运算数的值为假，表达式为第一个运算数的值，否则表达式为第二个运算数的值。

    逻辑或：||

    当||的第一个运算数的值为假时，表达式为第二个运算数的值，否则表达式为第一个运算数的值。

拦截网页的默认行为

linux

putty连接Linux中文乱码解决方案

在Linux中输入LANG=zh_CN.utf-8

点击Putty左上角的图标，找到Change Settings,再找到Window——>Translation——–>Remote character set，把它改为你设置的字符集，这里以UTF-8为例。

至此，大功告成，可以正常显示中文了。

Git

git 连接远程库。

首先要使用ssh-keygen 生成公钥，引号里输入自己的邮箱地址。

执行这个命令后可以一路回车，然后会生成一个私钥和一个公钥，公钥可以随便告诉别人，私钥一定不要让它泄密。Linux下默认保存在/root/.ssh下windows默认保存在C:\Users\window用户名\.ssh下。.ssh文件夹下会产生如下几个文件。

其中的id_rsa.pub就是公钥,在文本编辑器打开，复制里面的内容，在你的github账号下选择Settings，然后选择SSH and GPG keys，点击new ssh key,粘贴进去就可以使用git clone git@github.com:github用户名/远程仓库名.git 愉快地将远程仓库克隆下来了。
